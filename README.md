# API Task

This is an API CRUD for a list of tasks.

## Task structure

This tasks have this structure

```json
id: int
title: string
done: bool 
```


## API Documentation

The API is documented using Swagger.

After starting the server, open the following URL in your browser:

```
http://localhost:3000/api-docs
```

## Prerequisites

- Node.js


## Running

To run the app:

- clone the repository

```bash
git clone <url>
```

and then, in the terminal:

```bash
npm install
npm start
```
